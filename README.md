# Sudanese Mythology (Vite + React)

A simple website that collects short articles inspired by Sudanese mythology and folklore.

## Features

- Home page + About page
- Sudanese Mythology page with:
  - Search
  - Category filter (Desert / River / Markets)
- Article pages with dynamic routes (`/article/:id`)
- Clean navigation with active links

## Tech Stack

- Vite
- React
- TypeScript
- React Router

## Getting Started

```bash
npm install
npm run dev
```
